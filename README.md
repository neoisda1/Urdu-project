# بچوں کی اردو کہانیاں
## Urdu Short Stories for Young Children

یہ پروجیکٹ چھوٹے بچوں کے لیے اردو کہانیوں کا مجموعہ ہے جو الفاظ سیکھنے اور پڑھنے کی مشق کے لیے بنایا گیا ہے۔

This project is a collection of Urdu stories for young children designed to increase vocabulary and improve reading skills.

## استعمال کی ہدایات / Usage Guide

### 🌟 بہترین طریقہ: HTML Viewer استعمال کریں
**Recommended: Use the HTML Viewer**

`story-viewer.html` فائل کو براؤزر میں کھولیں:
Open the `story-viewer.html` file in a browser:

✨ **خصوصیات / Features:**
- 🔤 **خط کا سائز بڑھائیں یا کم کریں** (Font size control: 12px - 36px)
- 📖 آسانی سے کہانیاں تبدیل کریں (Easy story switching)
- 🎨 خوبصورت انٹرفیس (Beautiful interface)
- ⌨️ **Keyboard Shortcuts:**
  - `Ctrl/Cmd + +` = خط بڑا کریں (Increase font)
  - `Ctrl/Cmd + -` = خط چھوٹا کریں (Decrease font)
  - `Ctrl/Cmd + 0` = دوبارہ سیٹ (Reset font)

### ۱۔ پہلے سبق سے شروع کریں
Start with the introductory lesson: `00-سبق-پڑھنا-سیکھیں.md`
- اردو حروف سیکھیں
- حروف کو جوڑنا سیکھیں
- آوازیں پہچانیں

### ۲۔ پھر کہانیاں پڑھیں
Then read the stories in order:
- ہر کہانی آسان الفاظ میں ہے
- نئے الفاظ کی فہرست دی گئی ہے
- اخلاقی سبق شامل ہے
