# کہانی نمبر ۱۶: ننھا پودا
## Story #16: The Little Plant

---

ایک ننھا پودا تھا۔
وہ مٹی میں اگ رہا تھا۔

روز سورج نکلتا۔
بارش ہوتی۔

پودا آہستہ آہستہ بڑا ہوا۔
وہ ایک خوبصورت پھول بن گیا۔

سب نے پھول کو دیکھا اور مسکرائے۔

## سبق / Moral Lesson

**صبر اور محنت سے کامیابی ملتی ہے۔ ہر چیز کو وقت چاہیے۔**

Success comes with patience and effort. Everything needs time.

جیسے پودا آہستہ آہستہ بڑھتا ہے، ہم بھی آہستہ آہستہ سیکھتے اور بڑھتے ہیں۔

## کہانی کی سرگرمی / Story Activity

### ۱. پودا کیسے بڑھتا ہے؟ / How Does a Plant Grow?

پودے کی ترتیب:
Sequence of plant growth:

```
۱. بیج مٹی میں → 🌱
۲. پانی اور سورج → ☀️💧
۳. پتے نکلتے ہیں → 🌿
۴. پھول کھلتا ہے → 🌸
```

### ۲. پودے کی ضرورتیں / <span dir="ltr">Plant's Needs:</span>

پودے کو کیا چاہیے؟
<span dir="ltr">What does a plant need?</span>

```
✓ مٹی (Soil)
✓ پانی (Water)
✓ سورج کی روشنی (Sunlight)
✓ ہوا (Air)
✓ محبت (Love/care)
```

### ۳. باغبانی کریں / <span dir="ltr">Gardening:</span>

کیا آپ بیج لگائیں گے؟ کیا کریں گے؟
<span dir="ltr">Will you plant a seed? What will you do?</span>

```
☐ مٹی تیار کرو
☐ بیج بوؤ
☐ پانی دو
☐ دھوپ میں رکھو
☐ روز دیکھو
```

## خاص نظم / Special Poem

```
ننھا پودا مٹی میں
دیکھو کیسے اگتا ہے
پانی پیتا، دھوپ لیتا
آہستہ آہستہ بڑھتا ہے

سبز پتے لہراتے ہیں
رنگ برنگے پھول کھلے
دیکھ کر سب مسکراتے ہیں
خوشیاں چاروں طرف پھیلیں
```
