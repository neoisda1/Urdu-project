# کہانی نمبر ۲۴: ٹوپی اُڑ گئی
## Story #24: The Hat Flew Away

---

سارہ نے ٹوپی پہنی۔
ہوا چلی۔

ٹوپی اُڑ گئی!
سارہ نے دوڑ لگائی۔

ٹوپی درخت پر تھی۔
اب ٹوپی بھی ہنس رہی تھی 😄

---

## نئے الفاظ / New Vocabulary

| اردو | اردو (رومن) | English |
|-----|------------|---------|
| **ٹوپی** | Topi | Hat/cap |
| **ہوا** | Hawa | Wind |
| **اُڑنا** | Urna | To fly |
| **دوڑنا** | Daudna | To run |
| **درخت** | Darakht | Tree |
| **پہننا** | Pehenna | To wear |
| **چلنا** | Chalna | To blow/move |
| **لگانا** | Lagana | To apply/do |
| **ہنسنا** | Hansna | To laugh |

---

## سبق / Moral Lesson

**ہوا تیز ہو تو چیزیں اُڑ سکتی ہیں۔ مگر یہ مزے کی بات ہے!**

When wind is strong, things can fly. But it's fun!

قدرت کی چیزیں جیسے ہوا بعض اوقات حیرت انگیز اور مزے دار ہوتی ہیں۔

---

## سمجھنے کے سوالات / Comprehension Questions

۱. سارہ نے کیا پہنا؟
   - <span dir="ltr">What did Sarah wear?</span>

۲. کیا چلی؟
   - <span dir="ltr">What blew?</span>

۳. ٹوپی کو کیا ہوا؟
   - <span dir="ltr">What happened to the hat?</span>

۴. ٹوپی کہاں تھی؟
   - <span dir="ltr">Where was the hat?</span>

---

## کہانی کی سرگرمی / Story Activity

### ۱. ہوا والی چیزیں / <span dir="ltr">Windy Things:</span>

ہوا میں کیا اڑتا ہے؟
<span dir="ltr">What flies in the wind?</span>

```
✓ ٹوپی (Hat)
✓ پتنگ (Kite)
✓ پتے (Leaves)
✓ کاغذ (Paper)
✓ غبارہ (Balloon)
```

### ۲. آواز / Sound:

ہوا کی آواز:
Wind sound:

**ہو ہو ہو** 💨

### ۳. تصویر / Picture:
