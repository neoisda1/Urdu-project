# کہانی نمبر ۱۷: ننھا بلی کا بچہ
## Story #17: Tiny Kitten

---

ایک ننھا بلی کا بچہ تھا۔
وہ سفید تھا۔

بلی کا بچہ کھیل رہا تھا۔
وہ گیند کے پیچھے بھاگا۔

گیند رک گئی۔
بلی کا بچہ بھی رک گیا۔

وہ خوش تھا۔

---

## نئے الفاظ / New Vocabulary

| **گیند** | Gend | Ball |
| **کھیل** | Khel | Play/game |
| **بھاگنا** | Bhaagna | To run |
| **رکنا** | Rukna | To stop |
| **پیچھے** | Peechay | Behind |

| **خوش** | Khush | Happy |

---

## سبق / Moral Lesson

**کھیلنا اچھا ہے۔ کھیل سے ہم خوش ہوتے ہیں۔**

Playing is good. Games make us happy.

بچے کھیل کر خوش ہوتے ہیں اور صحت مند رہتے ہیں۔

---

## سمجھنے کے سوالات / Comprehension Questions

۱. بلی کا بچہ کیسا تھا?
   - <span dir="ltr">How was the kitten?</span>

۲. وہ کس رنگ کا تھا؟
   - <span dir="ltr">What color was it?</span>

۳. وہ کیا کر رہا تھا؟
   - <span dir="ltr">What was it doing?</span>

۴. وہ کیسا تھا؟
   - <span dir="ltr">How did it feel?</span>
