# کہانی نمبر ۱۲: مہربان نانبائی
## Story #12: The Kind Baker

---

شہر میں ایک نانبائی تھا۔ اس کا نام اکرم چچا تھا۔

وہ روز تازہ روٹیاں بناتا تھا۔ روٹیوں کی خوشبو دور تک جاتی تھی۔

ایک دن ایک غریب لڑکا دکان کے باہر کھڑا تھا۔ وہ بھوکا تھا۔

اکرم چچا نے اسے دیکھا۔ انہوں نے لڑکے کو گرم روٹی دی۔

لڑکے نے کہا، "میرے پاس پیسے نہیں۔"

اکرم چچا مسکرائے۔ "کوئی بات نہیں، بیٹا۔ روز آنا۔"

اب وہ لڑکا بڑا ہو کر اچھا انسان بنا۔ وہ دوسروں کی مدد کرتا ہے۔

## سبق / Moral Lesson

**بھوکے کو کھانا کھلانا بہت بڑی نیکی ہے۔ مہربانی کا پھل میٹھا ہوتا ہے۔**

Feeding the hungry is a great virtue. Kindness bears sweet fruit.

جو لوگ دوسروں کی مدد کرتے ہیں، اللہ ان کی مدد کرتا ہے۔

## سمجھنے کے سوالات / Comprehension Questions

۱. نانبائی کا کیا نام تھا؟
   - <span dir="ltr">What was the baker's name?</span>

۲. وہ کیا بناتا تھا؟
   - <span dir="ltr">What did he make?</span>

۳. لڑکا کہاں کھڑا تھا؟
   - <span dir="ltr">Where was the boy standing?</span>

۴. اکرم چاچا نے کیا دیا؟
   - <span dir="ltr">What did Uncle Akram give?</span>

۵. لڑکا بڑا ہو کر کیا بنا؟
   - <span dir="ltr">What did the boy become when he grew up?</span>

## کہانی کی سرگرمی / Story Activity

### ۱. نیکیوں کی فہرست / <span dir="ltr">List of Good Deeds:</span>

ہم دوسروں کی کیسے مدد کر سکتے ہیں؟
<span dir="ltr">How can we help others?</span>

```
✓ بھوکے کو کھانا دینا
✓ بیمار کی خبر لینا
✓ بزرگوں کا احترام کرنا
✓ چھوٹوں سے پیار کرنا
✓ راستہ دکھانا
```

### ۲. پیشے / Professions:

مختلف پیشوں کے نام:
Different professions:

| پیشہ | کام |
|------|-----|
| نانبائی | روٹی بناتا ہے |
| استاد | پڑھاتا ہے |
| ڈاکٹر | علاج کرتا ہے |
| کسان | فصل اگاتا ہے |

### ۳. اپنا تجربہ / <span dir="ltr">Your Experience:</span>

کیا آپ نے کبھی کسی کی مدد کی ہے؟ کیسے؟
<span dir="ltr">Have you ever helped someone? How?</span>
